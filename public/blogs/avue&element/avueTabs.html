    avue-tabs手动触发tab切换方法

    JinJuntong 3/1/2022 avue

    ​ avue 文档并未讲解手动触发 tabs 变换的方法，今天遇到个需求特定条件下进入该界面，tab 栏切换至第三栏，记录一下方法。

    this.refs.tabs.changeTabs(index)

    3.尝试方法

      监听路由参数是否带有isFromMsg, 是的话切换至第三个tab

    watch: {
    $route: {
        handler(newVal) {
            if (newVal) {
                const { isMessage } = newVal.params;
                if (!isMessage) return;
                this.$nextTick(() => {
                   this.$refs.tabs.changeTabs(2);
                });
            }
        },
        deep: true,
        immediate: true
    }
},
