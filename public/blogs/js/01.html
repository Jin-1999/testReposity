    # 4. Array.forEach

    forEach() 方法对数组的每个元素执行一次给定的函数。

    返回 undefined

    语法：arr.forEach(callback(currentValue [, index [, array]])[, thisArg])

    var arr = [1,,2,3,4]
let newarr = arr.forEach((v,i) => {
    v +=1;
    console.log(v)
})
// 2 3 4 5
console.log(newarr) //undefined

    # 5. Array.map

    map() 方法创建一个新数组，其结果是该数组中的每个元素是调用一次提供的函数后的返回值。

    语法：var new_array = arr.map(function callback(currentValue[, index[, array]]) { // Return element for new_array }[, thisArg])

    返回：一个由原数组每个元素执行回调函数的结果组成的新数组。

    若是单纯遍历数组且不返回新数组，应该但是Array.forEach而不是Array.map

    const arr = [1, , 2, 3, 4]
const newarr = arr.map(v => {
    return v = v * 2
})
console.log(newarr); // [2, <1 empty item>, 4, 6, 8]

    # 6. Array.some

    语法：array.some(function(currentValue,index,arr),thisValue)

    var arr = [1,,2,3,4]
arr.some((v,i) => {
    console.log(v)
})
// 1 2 3 4
    1
    2
    3
    4
    5

    some() 方法用于检测数组中的元素是否满足指定条件（函数提供）

    some()方法会依次执行数组的每个元素：

    • 如果有一个元素满足条件，则表达式返回true, 剩余的元素不会再执行检测。
    • 如果没有满足条件的元素，则返回false。
    // 判断数组是否有一项大于3， 有就跳出循环
var arr = [1, 2, 3, 4]
const fn = (v) => {
    return v > 3;
}
console.log(arr.some(fn));

    与Array.some用法一致，但是会遍历所有项，不会中途满足条件就停止。
