## 1、使用 vite+pnpm 初始化 vue 项目

```js
pnpm create vite my-vue-app -- --template vue
```

## 2、代码检查工具 eslint 安装和初始化

```js
pnpm add eslint -D
pnpm eslint --init
```

在 init 后自动生成 .eslintrc.js 配置文件
